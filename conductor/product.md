# Product Goals: HomeLabAI & Field Notes

## Vision
To build a resilient, high-fidelity hybrid local LLM ecosystem that integrates 18 years of technical history into a cohesive "Technical Dashboard" and agentic reasoning engine.

## Core Features
- **Bicameral Mind**: Hybrid orchestration between Linux (Pinky - RTX 2080 Ti) and Windows (The Brain - RTX 4090).
- **Poor Man's MoE (PMM)**: Dynamic LoRA expert routing on a unified Llama-3.2-3B base model to optimize VRAM.
- **Static Synthesis**: A pipeline (Librarian/Nibbler) that distills raw engineering notes into a searchable, high-fidelity static site.
- **Voice Gateway**: Persona-driven voice assistant architecture (Pinky/The Brain).
- **Fidelity Gate**: Autonomous verification of agent output density and technical alignment.

## Strategic Objectives
- Transition from passive "Tagging" to active "Harvesting" of high-fidelity technical depth.
- Bridge the gap between immediate reflex responses and deep forensic archive searches.
- Ensure 100% word-faithful technical narrative assets (Content Immutability).
